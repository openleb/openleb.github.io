the nouwweb project

nouwweb is a repository of data and tools for Lebanon and particularly the Lebanese parliament.

Featuring work by Lamba Labs, SMEX and Democracy International.

Help out:

  • fix open issues or share new ideas!
  • gather data
  • or report inaccuracies in our data
  • use our tools to build your own project! (and tell us about it)

Design adapted from dccode.org and theunitedstates.io.

/districts

Detailed data on administrative districts.

/names

Names of the parliamentary members and their info.

/search

Documentation about the search API for members of parliament.
