the @openleb project

openleb.io is a shared commons of data and tools for Lebanon made by the public for use by the public.

Featuring work by Lamba Labs, SMEX and the Internet.

Help out:

  • fix open issues or share new ideas!
  • start or submit a project for inclusion on this page
  • gather or report inaccuracies in the data
  • use our tools to build your own project! (and tell us about it on the mailing list)

Design adapted from dccode.org and theunitedstates.io.

/districts

Detailed data on administrative districts.

/nouwweb

Names of the parliamentary members and their info.

/search

An API to search for members of parliament
